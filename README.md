# About the Project
This is my entry into the Eta Kappa Nu Summer Chess Challenge.

Is it strange to think about assembly language and think "hey, that sounds fun"? Masochistic maybe? I've had a brush with Commodore 64 assembly in high school, and since then I've been struck by a desire to go farther with it (I only got as far as a "guess the number" program in the time I had). Why not try to make a full-featured chess program for the C64? Or try to make a chess engine that's as small as possible? Being so close to the hardware, the possibilities are limitless.

Of course, there'll be some steps between what I'm trying to do and the TUI chess program envisioned by the Chess Challenge, but with the right emulator setup I'm sure I can get something to work.

More to follow!

# About the Summer-2026-Chess-Challenge
The Eta Kappa Nu Summer Chess Challenge is a learning-first build challenge for friends who want to sharpen their programming skills, explore language design tradeoffs, and create fun content for the HKN GitHub and YouTube presence.

Each participant will build a chess platform in a different programming language, document their work throughout the summer, and optionally extend their project into networking and engine play.
